# VNM Job Portal - Backend API

This is the backend server for the **VNM Job Portal**, built using **Node.js, Express, and MongoDB**. It supports the Web Dashboard (for Employers/Candidates) and the Mobile App (for Candidates).

## 🚀 Tech Stack

* **Runtime:** Node.js & Express.js
* **Database:** MongoDB Atlas (Mongoose)
* **File Storage:** AWS S3 (Multer S3)
* **Authentication:** JWT (JSON Web Tokens)
* **Deployment:** AWS Elastic Beanstalk

---
